🔒 Données & Confidentialité

Données collectées

Ce site ne collecte aucune donnée personnelle des visiteurs. Aucun cookie de tracking, aucune publicité, aucun formulaire d'inscription.

Stockage local (localStorage)

Pour améliorer votre expérience, le navigateur mémorise localement :

  • Vos ressources favorites (identifiants, stockés dans votre navigateur uniquement)
  • Vos préférences de filtres et de tri

Ces données ne quittent jamais votre appareil et peuvent être effacées en vidant le cache de votre navigateur.

Base de données

Les ressources pédagogiques sont stockées sur Supabase (infrastructure cloud sécurisée, serveurs en Union Européenne). Seul l'administrateur du site peut créer, modifier ou supprimer des ressources, après authentification sécurisée.

Polices & ressources externes

Ce site utilise la police Plus Jakarta Sans via Google Fonts (requête HTTPS). La bibliothèque JavaScript Supabase Client est chargée via jsDelivr (CDN).
